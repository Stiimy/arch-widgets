# Sylph Widgets

Quickshell desktop widgets for Hyprland — crafted from [ilyamiro's dots](https://github.com/ilyamiro/imperative-dots), adapted for standalone use on Arch/HyDE.

## Widgets

| Keybind | Widget | Features |
|---------|--------|----------|
| `SUPER + N` | **Control Center** | Battery gauge, brightness slider, volume slider, live notification history |
| `SUPER + R` | **Network** | WiFi scan & connect, Bluetooth scan/pair/connect, Ethernet status, refresh button |
| `SUPER + O` | **Calendar** | Monthly calendar, weather forecast (OpenWeather API), day navigation |

All widgets close with `Escape` and reopen with the same keybind.

## Dependencies

```bash
yay -S quickshell-git iw
```

Required system packages:
- `quickshell-git` — QtQuick desktop shell framework
- `iw` — WiFi tools
- `nmcli` — NetworkManager CLI
- `bluetoothctl` — Bluetooth control (requires `bluez-utils`)
- `brightnessctl` — Backlight control
- `wpctl` — PipeWire/WirePlumber audio
- `jq` — JSON parsing
- `bash` — Script runner

Arch users:
```bash
sudo pacman -S networkmanager bluez-utils brightnessctl wireplumber jq
```

## Installation

```bash
# 1. Install quickshell
yay -S quickshell-git iw

# 2. Clone the widgets
git clone https://github.com/Stiimy/sylph-widgets.git ~/.config/quickshell

# 3. Add keybinds to ~/.config/hypr/keybindings.conf
cat >> ~/.config/hypr/keybindings.conf << 'EOF'
bind = $mainMod, N, exec, ~/.config/quickshell/battery-widget/launch.sh
bind = $mainMod, R, exec, ~/.config/quickshell/network-widget/launch.sh
bind = $mainMod, O, exec, ~/.config/quickshell/calendar-widget/launch.sh
EOF

# 4. Reload Hyprland
hyprctl reload
```

## Weather Setup (Calendar)

Edit `calendar-widget/.env` with your OpenWeather API key:
```env
OPENWEATHER_KEY=your_32_char_api_key
OPENWEATHER_CITY_ID=2988507
OPENWEATHER_UNIT=metric
```
Get a free key at [openweathermap.org](https://openweathermap.org/).
